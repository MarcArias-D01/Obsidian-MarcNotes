## Crear lista
```python
mi_lista = ["1","2","3","Hola lector"]
mi_lista = list("hola") # ['h', 'o', 'l', 'a']
mi_lista = list(range(5)) # [0, 1, 2, 3, 4]
```

# Funciones de lista
```python
mi_lista.append(x) # Añadir valro al final de la lista
mi_lista.extend([4, 5]) # Agregar elementos a la lista, sumar lsita
mi_lista = mi_lista + [4,5] # Agregar elementos a la lista, sumar lsita
mi_lista.insert(idx, x) #  inserta en una posición concreta
```
